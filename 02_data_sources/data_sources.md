## Data Sources

This project uses Olist’s publicly available marketing funnel datasets from Kaggle.

### Datasets Used
- olist_marketing_qualified_leads_dataset.csv  
- olist_marketing_closed_deals_dataset.csv  

### Time Period
2017–2018

### Notes on Data Quality
- CRM-style data with inconsistent data types
- Mixed representations for booleans and numeric attributes
- Required staged ingestion to ensure correctness
